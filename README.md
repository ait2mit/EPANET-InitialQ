# EPANET-InitialQ
1.	This package contains python codes to generate MSX file with calculated initial quality 
2.	This codes have been developed for personal use. If anybody want to use, there is no guarantee or implied guarantee that the codes outcomes are correct. If you want use please use on your own risk.
3.	Goal: Reduce Simulation Time for extra-large water network 
4.	Objectives: To configure EPANET-MSX file based on intermediate results so that simulation can be run for a stable network based on calculated initial concentration
5.	Dependency: This module depends on other two modules (i) extractNodeLinkList, (ii) EN_Results. This two modules must be stored in the environment
6.	Entry point of the package: InitQFormat.py
